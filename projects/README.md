# Projects

Dated, self-contained micro-projects — one folder per artifact, `YYYY-MM-DD_<slug>/`.

Each folder is independent: its own `README.md`, `LICENSE`, source, and tests. Each was built clean-room (a paper's method, never the authors' code), tested green, and passed the external policy + maker-checker gate before publication. None is a flagship — deep work graduates to its own top-level repository.

Reimplementations of published work are **independent implementations**, cited and not affiliated with or endorsed by the original authors.
